### 1.3 （可选）添加精彩点评服务器

箭头音乐支持查看歌曲的热门评论。目前只有箭头音乐支持此功能。

打开箭头音乐 → 右上角设置 → **精彩点评** → **编辑点评服务器** → **添加**：

```
备注:   随意（如"网易云评论"）
地址:   https://comment.str.ccwu.cc
用户名: 留空
密码:   留空
```

保存后，任意歌曲的播放页面点击评论按钮即可看到热门评论。

## 二、下载新歌（LX 音乐搜索）

当你发现想听的歌库里没有时，通过 LX 音乐搜索在线下载。

### 2.1 网页端（推荐）

打开浏览器访问 `https://lx.str.ccwu.cc/music`：

   0.密码为114514

1. 搜索框输入歌名
2. 在结果里点击歌曲 → **下载**
3. 下载完成后，歌曲自动进入道理鱼音乐库

> 💡 **安装为桌面 App**：用 Chrome / Edge 打开后，地址栏右侧会出现「安装」图标（⊕），点击即可生成独立的桌面应用。iPhone 上用 Safari 打开 → 分享 → 添加到主屏幕。Android Chrome → 菜单 → 添加到主屏幕。

### 2.2 电脑端（LX Music Desktop）

1. 下载 [LX Music Desktop](https://github.com/lyswhut/lx-music-desktop/releases)
2. 设置 → 音源 → 导入自定义音源（找管理员要音源文件）
3. 搜歌 → 下载，下载的歌曲会自动同步到服务器

### 3.2 强烈推荐：SPlayer（全平台）

> SPlayer 是第三方网易云客户端，额外支持 Subsonic 协议接入流媒体服务器。适合喜欢听在线音乐的人。

- 下载：[SPlayer]([下载 | SPlayer](https://splayer.imsyy.top/download.html))
- 平台：Windows / macOS / Linux
- 亮点：界面精美，体验均衡，开箱即用，听 Subsonic 服务器最省心的选择（目前似乎不支持道理鱼音乐，只能先用subsonic，等主包魔改）<img src="C:\Users\Administrator\AppData\Roaming\Typora\typora-user-images\image-20260525233158426.png" alt="image-20260525233158426" style="zoom: 25%;" />

### 3.3 强烈推荐：ECHO（Windows / Linux）

> 面向本地流媒体曲库、HiFi 输出和长期维护的开源桌面音乐播放器。适合偏好管理本地下载音乐文件、追求 HiFi 输出的用户。

- 下载：[ECHO](https://github.com/Moekotori/ECHO)
- 平台：Windows / Linux（不支持 macOS）
- 特色功能：自动搜索播放歌曲 MV 作为背景，功能丰富，专注于本地流媒体曲库管理<img src="C:\Users\Administrator\AppData\Roaming\Typora\typora-user-images\image-20260525235231144.png" alt="image-20260525235231144" style="zoom: 25%;" />

### 3.4 推荐：GMPlayer（全平台，需自行编译）

> SPlayer 的增强版，功能更丰富。有一定技术门槛，需要自行编译，非普通用户可开箱即用。

- 下载：[GMPlayer](https://github.com/Burial0268/GMPlayer)
- 平台：Windows / macOS / Linux
- 注意：需要自己编译，目前仅有网页demo，适合有一定开发经验的用户

## 四、常见问题

### Q: 主包主包在哪下载😡
在群里有喵😭

### Q: 为什么我用的客户端没有道理鱼💀

目前似乎只有箭头音乐支持哦，别的客户端可以用subsonic服务器，两者音乐库是一样的😋

### Q: LX 音乐搜索下载的歌多久能听到？🤔
下载完成后会自动进入道理鱼，通常在 **30 秒到 3 分钟**内扫描入库。着急的话让管理员手动触发扫描。😊

### Q: 手机流量能听吗？😈
取决于服务器配置。如果开启了公网访问，用流量也能听（注意流量消耗）。**建议在 WiFi 下听无损音质**。🥰

### Q: 如何获取账号？🤓
联系管理员创建。🥹
